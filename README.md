# Bubble Sort Program in Python

## 📌 Description
This is a simple Python program that takes user input to create a list of numbers and sorts them using the Bubble Sort algorithm.

The program:
- Asks the user how many numbers they want to enter
- Accepts the numbers one by one
- Displays the unsorted list
- Sorts the list using Bubble Sort (with optimization)
- Displays the sorted list

## ⚙️ Algorithm Used
Bubble Sort with an optimization:
- Repeatedly compares adjacent elements
- Swaps them if they are in the wrong order
- Stops early if no swaps occur in a pass (efficient for sorted lists)

## ▶️ How to Run
1. Run the Python file
2. Enter the number of elements
3. Input the numbers when prompted
4. View the sorted result
